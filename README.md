# 上门洗鞋·在线下单

一个简单的 Web 工具，包含客户下单页和商家后台管理页。

## 在线演示

- 客户下单页：https://chengyuxia.com/washv5/customer.php
- 商家后台：https://chengyuxia.com/washv5/merchant.php

## 功能

- **客户下单**：填写信息，提交订单
- **商家后台**：查看今日/本月收入、订单列表、更新订单状态

## 技术栈

- PHP
- 原生实现，无框架

## 使用说明

将 `washv5` 文件夹部署到支持 PHP 的服务器即可。
